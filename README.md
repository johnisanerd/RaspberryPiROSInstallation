# RaspberryPiROSInstallation
Install ROS on the Raspberry Pi

Following instructions on [Installing ROS Kinetic on the Raspberry Pi.](http://wiki.ros.org/ROSberryPi/Installing%20ROS%20Kinetic%20on%20the%20Raspberry%20Pi)

Burn Raspbian Stretch with Etcher.
Setup SSH - Add a text file and rename it "ssh"
Rename to "rosbot"  You can now login at rosbot.local, with the username "pi" and the password "raspberry"
